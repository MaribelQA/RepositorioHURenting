CA-001: Búsqueda de vehículos disponibles
========================================================

CP-001
Título         : Búsqueda exitosa con parámetros válidos
CA             : CA-001
Descripción    : Verificar que el sistema devuelve vehículos disponibles
                 cuando se ingresan fecha de inicio, fecha de fin y ciudad válidas.
Precondiciones : - El usuario está autenticado en el sistema.
                 - Existen vehículos registrados y disponibles en la ciudad "Bogotá"
                   para las fechas 2026-04-01 al 2026-04-05.
Pasos          :
  1. Ingresar a la página principal del sistema de Renting.
  2. Seleccionar la ciudad de recogida: "Bogotá".
  3. Ingresar fecha de inicio: 2026-04-01.
  4. Ingresar fecha de fin: 2026-04-05.
  5. Hacer clic en el botón "Buscar".
Resultado Esp. : El sistema muestra una lista de vehículos disponibles para las
                 fechas y ciudad indicadas.
                 Cada resultado incluye: nombre del vehículo, categoría y precio por día.
Tipo de Prueba : Funcional
Prioridad      : Alta
Estado         : Activo

CA-008: Cancelación de reserva
========================================================

CP-023
Título         : Cancelar una reserva con más de 24 horas de anticipación
CA             : CA-008
Descripción    : Verificar que el cliente puede cancelar una reserva con anticipación
                 mayor a 24 horas y que el sistema actualiza el estado correctamente.
Precondiciones : - El cliente tiene una reserva "Confirmada" con fecha de recogida
                   en más de 24 horas.
                 - El usuario está autenticado.
Pasos          :
  1. Acceder a "Mis Reservas" en el perfil.
  2. Seleccionar la reserva a cancelar.
  3. Hacer clic en "Cancelar Reserva".
  4. Leer la política de cancelación mostrada.
  5. Confirmar la cancelación.
Resultado Esp. : - Se muestra la política de cancelación antes de confirmar.
                 - El estado de la reserva cambia a "Cancelada".
                 - El vehículo queda disponible para nuevas reservas.
                 - Se envía un correo de confirmación de cancelación al cliente.
Tipo de Prueba : Funcional
Prioridad      : Alta
Estado         : Activo
